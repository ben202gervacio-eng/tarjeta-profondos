Flujo de trabajo

  1. Edita los valores arriba y haz clic en "Generar codigo"
  2. Copia el codigo generado
  3. Abre index.html y reemplaza la seccion CONFIG con el nuevo codigo
  4. En la terminal: git add . → git commit -m "actualiza config" → git push
  5. Netlify lo despliega automatico en 1-2 minutos
